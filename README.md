# Usage
```
python3 readr-1.0.py -i <inputFolder> -o <outputFolder> -t <threshhold> -r <override> -d
```

* "-i", "--input"

folder for source data

* "-o", "--output"

folder for output

* "-t", "--threshhold"

thresh hold

* "-r", "--override": True or False

override the output files, default is Ture

* "-d", "--debug"

enable debug output
